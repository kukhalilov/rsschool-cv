Code example

Some numbers have funny properties. For example: 695 --> 6² + 9³ + 5⁴= 1390 = 695 * 2 or 46288 --> 4³ + 6⁴+ 2⁵ + 8⁶ + 8⁷ = 2360688 = 46288 * 51. Given a positive integer n written as abcd... (a, b, c, d... being digits) and a positive integer p. We want to find a positive integer k, if it exists, such that the sum of the digits of n taken to the successive powers of p is equal to k * n. If it is the case we will return k, if not return -1. 

                
function digPow(n, p) {
    digits = [...(n + "")].map((c) => +c);
    sum = 0;
    for (let digit of digits) {
        sum += Math.pow(digit, p);
        p++;
    }
    k = sum / n;
    if (Number.isInteger(k)) {
        return k;
    } else {
        return -1;
    }
    }
